  3. Некоторые важные теги для работы с текстом и их применение:


    header - это шапка страницы (на этой странице это самая верхняя картинка, но может быть и текст)

    nav - используется для организации навигации либо меню

    dev - блочный тэг середины страницы

    footer - подвал страницы, самая нижняя часть

    a - ссылка, если прописать a target="_blank" - ссылка, которая откроется в новом окне (именно с помощью него ты можешь перейти по ссылке и загрузить редактор кода, а также посмотреть видео как настроить его в первом пункте на этой странице; все ссылки в "подвале")

    img - изображение которая вставляется из папки img (картинка в шапке, подвале и т.д.)

    h1...6 - заголовки разной величины, блочный тэг, причем h1 может быть только один (самый первый текст)

    p - абзац, блочный тэг для текста

    span - линейный тэг для текста

    br - тэг переноса на следующую строку

    ol - нумерованный тэг, блочный (именно с помощью него написаны пункты на этой странице)

    ul - маркерованный тэг, блочный

    li - тэг с текстом внутри ul или ol (текст, который ты сейчас читаешь)




  4. А теперь пример как можно организавать формы:


form - оболочка формы, передача данных на сервер

method - атрибут тэга form, отвечает за метод передачи данных:

input - тэг приема данных от пользователей

type - атрибут тэга input, бывает разных видов:

text (name value) - для ввода текста,логина

password (name value)- для ввода скрытых символов, например пароля (обе эти формы вы можете видеть под шапкой страницы)
